# Trainig session

Day and start time of when swim team has a trainig session and its duration.

## Representaion of trainig session

### Day

Any day from the week, when the trainig session occurs.

### Start time

A time in format `HH:MM` of when the trainig session begins.

### Duration

A duration of the trainig session in minutes.

## JSON representaion

```json
{
  "day": "monday",
  "startTime": "18:00",
  "durationMin": 60
}
```
