# Calculate distance

## Actors

- Coach or Swimmer

## Description

User wants to see how many meters were swam between two dates.

## Success Scenario

1. User selects start date
2. User selects end date
3. User submits the date range
4. User is shown how many meters were swam in between the start and end date

## Extensions

- 2A: End date is before start date
  1.  Highlight date fields
