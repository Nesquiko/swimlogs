# Display training in this week

## Actors

- Coach or Swimmer

## Description

Web UIs landing page displays all trainings in this week, even if they have been
already swimmed.

## Success Guarantees

User can see list of trainings in this week in web ui.

## Success Scenario

1. User opens website
2. Trainings are loaded
3. Trainings are displayed

## Extensions

- 2A Service is unavailable
  1.  Display "SwimLogs is not available now"
- 2B No trainings are yet created for this week
  1.  Display "No trainings for this week"
